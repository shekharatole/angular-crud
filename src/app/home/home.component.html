Evolent Health - Project exercise for Application Development Positions

Project Description:

This application is designed and implemented for maintaining contact information.

I have implemented this application using implement a Single Page Application - Angular 7 version, also created the dummy database to store the user list. Tried to implement the basic CRUD functinality using Angular.

Functionality:

  • User list
  • Add a new user
  • Edit specific user
  • Delete specific user
  • Active / Deactive user

User fields:

  • First name
  • Last name
  • Email
  • Mobile Number
  • Status (Active / Deactive) - Default status is set as Deactive
